💜 an ode to heroku

The brain is bring-your-own.

claude-opus-4-8 runs on Heroku Managed Inference — it does the one genuinely hard, creative thing here: read a sentence a human typed and write correct code for it. That happens once. Cloudflare stores what Heroku wrote and executes it on the edge on every event after, for milliseconds and no tokens. Author on Heroku; execute on the edge. Existing billing untouched, the model swappable behind AI Gateway — the compute just moves to where a made decision belongs.
